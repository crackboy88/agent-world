# Chen Company Agent World - UI界面通用规范

---

## 【UI视觉通用规范】

### 1. 风格定位

| 项目 | 规范 |
|------|------|
| 风格 | 写实精致的商务向2D像素风，与办公场景、Agent角色100%统一 |
| 主题 | 浅色白底商务风 |
| 适用场景 | 侧边栏、顶部栏、弹窗、提示等所有UI组件 |

### 2. 配色规范

#### 主色调

| 颜色名称 | 色值 | 用途 |
|----------|------|------|
| 主色 | #2563EB | 按钮、选中状态、强调元素 |
| 主色悬停 | #1D4ED8 | 按钮悬停状态 |
| 主色按下 | #1E40AF | 按钮按下状态 |

#### 辅助色

| 颜色名称 | 色值 | 用途 |
|----------|------|------|
| 深灰 | #1F2937 | 文字、边框 |
| 中灰 | #6B7280 | 次要文字 |
| 浅灰背景 | #F3F4F6 | 卡片背景、输入框 |
| 浅灰边框 | #E5E7EB | 边框、分隔线 |
| 白色 | #FFFFFF | 背景、输入框内文 |

#### 点缀色

| 颜色名称 | 色值 | 用途 |
|----------|------|------|
| 浅蓝 | #60A5FA | 链接、提示 |
| 绿色 | #10B981 | 成功状态 |
| 红色 | #EF4444 | 错误/警告状态 |
| 黄色 | #F59E0B | 警告状态 |

### 3. 网格规范

| 项目 | 规范 |
|------|------|
| 基础网格 | 8px 网格系统 |
| 元素尺寸 | 所有元素尺寸为 8px 的整数倍 |
| 间距 | 8px / 16px / 24px / 32px |
| 内边距 | 8px / 12px / 16px |
| 圆角 | 4px / 8px / 12px |

### 4. 像素表现要求

| 项目 | 规范 |
|------|------|
| 边缘处理 | 锐利无模糊，硬边缘像素块 |
| 抗锯齿 | 无抗锯齿溢出 |
| 高光带 | 带 1px 高光带增强立体感 |
| 投影 | 带 2px 柔和投影增强体积感 |
| 投影色 | #000000，透明度 10% |

## 【绝对禁止内容】

- ❌ 禁止使用非像素风格
- ❌ 禁止使用非指定配色
- ❌ 禁止元素尺寸不是8的倍数
- ❌ 禁止使用模糊/抗锯齿边缘
- ❌ 禁止悬停无反馈
- ❌ 禁止禁用状态可点击
- ❌ 禁止聚焦状态无视觉提示
- ❌ 禁止中英文混杂
