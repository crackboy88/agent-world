# Chen Company Agent World - Agent角色通用规范

---

## 【Agent视觉通用规范】

### 1. 视图与风格

| 项目 | 规范 |
|------|------|
| 视图类型 | 2D 45度俯视等轴测视角（Top-down Isometric） |
| 风格定位 | 写实精致的商务向2D像素风，与办公场景100%统一，细节精度对标《八方旅人》HD-2D像素角色 |
| 配色主题 | 与各Agent所属房间主色调协调，Agent着装体现职业特征 |
| 适用场景 | 地图行走、房间内工作、交谈、思考等 |

### 2. 基础尺寸与结构

| 项目 | 规范 |
|------|------|
| 单帧立绘尺寸 | 128×128px（统一标准） |
| 行走动画 | 8方向（上下、左上/右上、左下/右下、左右），每方向4帧 |
| 站立/工作动画 | 4帧循环（呼吸感） |
| 思考动画 | 8帧循环（转圈/闪烁） |
| 行走速度 | 中速，每帧间隔 150ms |
| 透明背景 | PNG-24 透明背景 |

### 3. 基础元素构成

| 元素 | 规范 |
|------|------|
| 人物立绘 | 128×128px主体，45度俯视角，商务着装 |
| 头顶名字 | 白色文字 + 深色半透明背景，圆角矩形，长度自适应 |
| 头顶技能标签 | Emoji图标 + 技能名，深色半透明背景，圆角小标签 |
| 脚下阴影 | 椭圆形，#000000，透明度 20%，尺寸 40×20px |
| 状态图标 | 20×20px，显示于立绘右上角 |
| 消息气泡 | 圆角矩形，弹出于立绘上方 |

### 4. 像素精度要求

| 项目 | 规范 |
|------|------|
| 基础网格 | 8px 系统 |
| 最小细节 | 4×4px（纽扣、眼神高光等） |
| 线条粗细 | 2px 轮廓线，深色 #333333 |
| 色彩数量 | 每角色不超过 12 种颜色 |
| 缩放适配 | 支持 0.5x / 1x / 1.5x / 2x 显示 |

## 【Agent状态显示通用规范】

### 1. 空闲状态（Idle）

| 项目 | 规范 |
|------|------|
| 触发条件 | Agent 无任务执行，处于等待状态 |
| 视觉表现 | 站立姿态，身体轻微上下浮动（呼吸感） |
| 动画参数 | 4帧循环，每帧 500ms |
| 浮动幅度 | 垂直方向 ±2px |
| 特效 | 无 |
| 状态图标 | 无 |

### 2. 工作中状态（Working）

| 项目 | 规范 |
|------|------|
| 触发条件 | Agent 正在执行任务 |
| 视觉表现 | 坐姿/站姿工作姿态，手部有操作动作 |
| 动画参数 | 4帧循环，每帧 300ms |
| 浮动幅度 | 无 |
| 特效 | 进度条显示于立绘下方 |
| 进度条 | 长度 60px，高度 6px，背景 #E0E0E0，进度 #4CAF50 |

### 3. 思考/等待状态（Thinking）

| 项目 | 规范 |
|------|------|
| 触发条件 | Agent 等待任务指令或正在思考 |
| 视觉表现 | 站立姿态，头顶显示思考气泡，手指轻点下巴 |
| 动画参数 | 8帧循环，每帧 200ms |
| 特效 | 头顶旋转省略号 "..." 或问号 "?" 气泡 |
| 气泡样式 | 圆角矩形，白色背景 + 灰色边框，大小 32×24px |
| 状态图标 | 💭 灰色 |

### 4. 新任务提示状态（New Task）

| 项目 | 规范 |
|------|------|
| 触发条件 | 收到新任务分配 |
| 视觉表现 | 站立姿态，身体闪烁发光 |
| 动画参数 | 闪烁频率 500ms 一次，闪烁 3 次后停止 |
| 特效 | 外发光，#FFD700 金色，透明度 30% |
| 气泡提示 | 任务名称气泡，黄色背景 #FFF9C4 |
| 气泡位置 | 立绘上方偏左 |
| 状态图标 | ✉️ 黄色 |

### 5. 对话/交谈状态（Chatting）

| 项目 | 规范 |
|------|------|
| 触发条件 | Agent 正在与其他Agent或用户对话 |
| 视觉表现 | 面对面站立姿态，面部朝向对话对象 |
| 动画参数 | 4帧循环，每帧 400ms |
| 特效 | 对话气泡显示 |
| 气泡样式 | 圆角矩形，白色背景，黑色文字 |
| 最大气泡宽度 | 120px，自动换行 |
| 状态图标 | 💬 蓝色 |

### 6. 移动状态（Walking）

| 项目 | 规范 |
|------|------|
| 触发条件 | Agent 正在房间之间移动 |
| 视觉表现 | 行走姿态，8方向，腿部交替运动 |
| 动画参数 | 4帧循环，每帧 150ms |
| 移动速度 | 100px/秒（中速） |
| 路径显示 | 地面显示虚线路径，#2196F3 蓝色 |
| 状态图标 | 无 |

### 7. 离线状态（Offline）

| 项目 | 规范 |
|------|------|
| 触发条件 | Agent 断开连接或未启动 |
| 视觉表现 | 站立姿态，灰色滤镜（饱和度 -80%） |
| 动画参数 | 无动态，保持静态 |
| 特效 | 整体灰度显示 |
| 状态标签 | "离线" 文字标签显示于立绘下方 |
| 状态图标 | ⭕ 灰色 |

### 8. 情绪变化系统

#### 8.1 积极情绪（Positive）

| 项目 | 规范 |
|------|------|
| 触发条件 | 任务完成、收到表扬、获得资源时 |
| 视觉表现 | 面部微笑，立绘颜色微暖（饱和度 +10%） |
| 特效 | 短暂爱心/星星飞出动画 |
| 持续时间 | 2秒后恢复常态 |

#### 8.2 消极情绪（Negative）

| 项目 | 规范 |
|------|------|
| 触发条件 | 任务失败、被批评、资源不足时 |
| 视觉表现 | 面部沮丧，立绘颜色微冷（饱和度 -10%） |
| 特效 | 短暂雨滴/叹号动画 |
| 持续时间 | 3秒后恢复常态 |

#### 8.3 中性情绪（Neutral）

| 项目 | 规范 |
|------|------|
| 触发条件 | 默认状态 |
| 视觉表现 | 面部平和，无特效 |

## 【绝对禁止内容】

### 1. 视觉禁止

- ❌ 禁止使用非像素风格（必须HD-2D像素风）
- ❌ 禁止超过 128×128px 尺寸
- ❌ 禁止非透明背景（PNG必须透明）
- ❌ 禁止与办公场景风格不统一
- ❌ 禁止过度游戏化形象（如卡通/二次元）

### 2. 状态禁止

- ❌ 禁止状态图标使用非标准Emoji
- ❌ 禁止进度条颜色使用非标准色
- ❌ 禁止离线状态显示彩色

### 3. 协作禁止

- ❌ 禁止线条使用非标准颜色
- ❌ 禁止箭头方向错误（必须从发起方指向接收方）
- ❌ 禁止协作线遮挡房间/家具

### 4. 动画禁止

- ❌ 禁止帧率低于 30fps
- ❌ 禁止闪烁频率过快（最快 200ms/次）
- ❌ 禁止移动速度过快或过慢（固定 100px/秒）
